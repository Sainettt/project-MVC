   1. Opis projektu:

Projekt zaliczeniowy "Aplikacja zarządzania przepisami na koktajle" ma na celu zweryfikowanie
umiejętności praktycznych studentów oraz sprawdzenie osiągniętych przez nich kompetencji w tworzeniu
aplikacji w środowisku Node.js z wykorzystaniem wzorca architektonicznego MVC. 
Projekt jest realizowany w formie repozytorium na platformie GitHub.


   2. Funkcjonalności:

Przeglądanie listy przepisów na koktajle
Dodawanie nowych przepisów
Edycja istniejących przepisów
Wyświetlanie szczegółowych informacji o przepisie
Ocena i komentowanie przepisów przez użytkowników

   3. Instrukcje obsługi:

 1. Uruchomienie aplikacji:
Sklonuj repozytorium z GitHuba: git clone <adres_repozytorium>
Przejdź do katalogu projektu: cd aplikacja-zarzadzania-przepisami
Zainstaluj zależności: npm install
Uruchom aplikację: npm start

 2. Instrukcje dodatkowe:
Aplikacja wykorzystuje następujące biblioteki zewnętrzne:
Express.js
Body-parser
EJS
